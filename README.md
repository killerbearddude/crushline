# Crushline Custom UI

A custom C++20 factory/node-dashboard UI built from low-level open-source tools.

Initial target:
- C++20
- CMake
- SDL3 later
- OpenGL 3.3 later
- Custom 2D renderer
- Custom retained-mode UI
- Custom node graph editor

## Build

Run:

    ./scripts/build.sh

## Run

Run:

    ./scripts/run.sh
